About This Project

The Quantum Harmonic Oscillator (QHO) is the most fundamental exactly-solvable quantum system. Its ground state energy is: E₀ = ½ℏω


This project implements the Variational Quantum Eigensolver (VQE) using IBM's qiskit-nature library to recover this energy computationally — and validates it against the exact analytical result. The project also includes a visual dashboard where users can adjust parameters (ℏω, shots, iterations) and run VQE without writing any code.

How to Run This Project

1️⃣

Clone the Repository

Download all project files to your computer

2️⃣

Install Dependencies

Run pip install to get all required packages

3️⃣

Launch Dashboard

Run Voilà to open the visual interface in your browser

# Step 1 — Clone the repository
git clone https://github.com/deeepakchauhan/qi26_21_proposal
cd qi26_21_proposal

# Step 2 — Install all required packages
pip install -r requirements.txt

# Step 3 — Launch the visual dashboard (no code needed)
voila QHO_Dashboard.ipynb

# OR: open the full analysis notebook
jupyter notebook QHO_Notebook.ipynb

ℹ The dashboard opens in your browser automatically. Use the sliders to adjust ℏω and shots, then click Run VQE to compute the ground state energy interactively.
